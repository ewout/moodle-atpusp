Menu

This theme uses a JavaScript drop-down menu system. To use this menu you must have JavaScrpt enabled in your web browser.

To login the user is presented with a login box in the header so that they can enter their username and password. Once logged in, the user is presented with three tabs. "Logout","My Courses" and "Update Profile" (these are only visible after login).

You can modify the links for the drop-down submenu list in mainmenu.php, which is commented so that you might understand the set up.
