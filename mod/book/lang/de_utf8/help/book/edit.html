Kapitel bearbeiten

Versuchen Sie die Überschriften der Kapitel knapp zu halten. Sie stehen links im Inhaltsverzeichnis. Lange Titel werden umbrochen und sehen nicht gut aus.

Neben der Hauptebene kann eine Unterkapitelebene angelegt werden.

Wenn Sie Probleme mit dem WYSIWYG HTML Editor haben, versuchen Sie einen anderen Browser z.B. Mozilla Firefox zu benutzen oder schalten Sie den Editor im Profil ab.
