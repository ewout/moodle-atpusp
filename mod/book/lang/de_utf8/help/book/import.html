Import

Sie können Buchseiten als HTML-Dateien importieren. Jede einzelne HTML-Datei wird zu einer neuen Seite des Buches. Die Anordnung erfolgt alphabetisch nach den Titeln der HTML-Dateien.

Relative Dateiverknüpfungen werden zu absoluten Kapitelverlinkungen konvertiert. Bilder, Flash und Java Inhalte werden ebenfalls verlinkt.
