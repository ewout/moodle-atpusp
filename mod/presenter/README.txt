Moodle Presentation Module v2 Release Notes
===========================================

Module version: 2009112602
Date: 26 nov 2009

Changes: 

	-> flv files are now played using FlowPlayer : http://www.flowplayer.org
	-> YouTube videos are now played using the customizable YouTube Player
	-> In the presentaion setup (add/edit), for audio (mp3) files, the start time can no longer be set up. The audio presentation will start from second 0 of the audio file.
