Tags

What are tags?

Tags are short pieces of text that can be used to categorise your blog posts. They are generally displayed in lower-case. Most tags are a single word, but you can also use multiple words if you like.

You enter tags into the Tags box when creating or editing a post. Tags are optional, so you can leave the box blank if you prefer.

Example: if you often write about your cat Fluffy, you might like to use the tag fluffy for these posts.

Multiple tags

You can include multiple tags in a post by separating them with commas. This can be useful if your post covers more than one topic.

Example: fluffy, global warming

Searching for tags

When you use tags in your posts, the Tags block appears on the main page of your block. This shows all the tags you've used. You can then click on a tag to see all the posts that contain that tag.

Example: a reader who is interested in global warming might click the global warming tag so that they can see all the posts about global warming in your blog, without needing to read too many posts about your cat.
