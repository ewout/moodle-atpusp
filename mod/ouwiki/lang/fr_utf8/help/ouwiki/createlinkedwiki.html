Créer des pages de wiki

Créer des pages de wiki est inhabituel si vous n'êtes pas familier des wikis.

  • Un principe-clé est que les pages d'un wiki doivent être liées d'une certaine façon.
  • Pour que ce soit toujours le cas, vous devez créer un lien vers la nouvelle page avant de pouvoir créer celle-ci.

Liens de wiki

Lorsque vous modifiez une page, des liens sont créés en tapant le titre d'une page à lier entre doubles parenthèses carrées.

  • Si vous dérirez créer un lien vers une page appelée Suisse, vous taperiez [[Suisse]].

Nouvelles pages

Pour créer une nouvelle page:

  1. Réfléchissez à un titre pour votre page. Ce titre doit des titres des autres pages du wiki. Il devrait décrire le contenu de votre page. Un titre pourrait par exemple être Grenouilles et autres amphibiens.
  2. Décidez quelle page devrait contenir un lien vers votre nouvelle page. Si le wiki est de taille réduite, ce pourrait être la page de démarrage. Sinon, trouvez une page appropriée.
  3. Modifiez la page qui contiendra le lien.
  4. Trouvez l'endroit où vous voulez voir apparaître le lien, et tapez-le dans le texte : [[Grenouilles et autres amphibiens]].
  5. Enregistrez la modification. Le lien créé doit apparaître, prêt à être utilisé.
  6. Cliquez sur le lien. Le système vous proposera alors de créer une nouvelle page.
  7. Cliquez Créer page. Une fenêtre d'édition apparaît alors pour votre nouvelle page.
  8. Ecrivez le texte initial de votre page, puis enregistrez-la. Votre page est maintenant créée.

Conseils

  • Si vous voulez créer plusieurs pages, vous voudrez peut-être créer tous les liens en une opération.
  • Vous pouvez créer des liens vers des pages, même si vous n'avez pas l'intention de les créer. Lorsque c'est le cas, les pages "manquantes" sont indiquées dans la vue de l'index du wiki. Ceci peut être utilisé pour indiquer que vous pensez que ces pages devraient exister.
  • Prenez garde aux titres - une fois une page créée, son titre ne peut plus être changé.
