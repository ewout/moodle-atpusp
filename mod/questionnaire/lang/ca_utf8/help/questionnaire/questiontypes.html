Tipus de preguntes

Seleccioneu el tipus de pregunta a partir de la llista del menú d'opcions Tipus sobre el botó "Afegir pregunta del tipus seleccionat". Cada tipus es mostrarà a la vostra enquesta de la següent forma:

---Salt de pàgina---

Utilitzeu aquest per inserir salts de pàgina dins enquestes llargues. Noteu que si una pàgina conté preguntes amb respostes obligatòries, la persona que respon no podrà passar a la següent pàgina si aquestes respostes obligatòries no s'han contestat.

Data
Utilitzeu el format dia/mes/any (p. ex. 14/3/1945, per al 14 de març de 1945)

Utilitzeu aquest tipus de pregunta si voleu que la resposta sigui una data correctament formatejada. El format dependrà del llenguatge utilitzat per la persona que respon l'enquesta. Per exemple, 4/21/2007 (EEUU); 21/4/2007 (UK); 21-4-2007 (França); etc. Un exemple ilustrat amb la data del dia actual es mostrarà a l'enquesta. Per poder exportar correctament les dates a fulls de càlcul com Excel, la persona que respon ha d'introduir una data entre 1902 i 2037. Si s'espera que les persones que responen introdueixin una data fora d'aquest rang, és millor utilitzar el tipus de pregunta Casella de text. El tipus de pregunta data acceptarà dates que consisteixin només en un mes i un any (per exemple, 12/2008 per desembre de 2008) o només un any (p. ex. 2008). No obstant, per motius de compatibilitat amb els fulls de càlcul, aquestes dates incompletes es convertiran automàticament en dates completes del tipus dd/mm/yyyy, per exemple, 12/2008 -> 01/12/2008 i 2008 -> 01/01/2008. És convenient explicar aquesta peculiaritat als usuaris de l'enquesta amb anterioritat.

Menú d'opcions

No hi ha cap avantatge real entre emprar el menú d'opcions respecte als botons d'opció, excepte per a llistes llargues d'opcions, per poder estalviar espai a la pantalla.

Numeric

Use this question type if you expect the response to be a correctly formatted number. By using the Max. digits allowed and Nb of decimal digits parameters you can specify the length and number of decimal places required.

Numèric

Utilitzeu aquest tipus de pregunta si voleu que la resposta sigui un número formatejat correctament. Amb els paràmetres Màxim de dígits permesos i Nombre de dígits decimals podeu especificar la llargària i el nombre de llocs decimals requerits.
