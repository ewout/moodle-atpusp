Visualització de la resposta

Podeu especificar qui pot veure les respostes de totes les persones a les enquestes enviades (taules d'estadístiques generals).

Els estudiants poden veure TOTES les respostes ->

  1. Mai . Per defecte, només el professorat del curs on està definida l'enquesta pot veure totes les respostes enviades de cada enquesta.
  2. Després de respondre l'enquesta . Això permet que els estudiants vegin totes les respostes després de respondre una enquesta que està configurada com respondre una vegada. Si una enquesta està configurada per ser contestada més d'una vegada, els estudiants no poden veure totes les respostes si no s'utilitza una de les següents dues opcions.
  3. Quan l'enquesta està tancada .
  4. Sempre.

Utilitzeu el sistema de rols i capacitats per decidir a quins participants en el curs es permetrà visualitzar a) les seves pròpies respostes b) totes les respostes.
