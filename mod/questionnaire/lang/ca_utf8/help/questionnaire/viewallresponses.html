Visualitzar totes les respostes - Filtre

Si l'enquesta està configurada amb Mode de Grup:Grups visibles, o està configurada a Grups separats, i l'usuari actual té definida la capacitat moodle/site:accessallgroups (en el context actual), i s'han definit grups al curs actual, llavors l'usuari té accés a una llista desplegable de grups. Aquesta llista desplegable permet que l'usuari "filtri" les respostes de l'enquesta per grups. Si la configuració és Mode de Grup: Grups separats, llavors els usuaris que no tinguin la capacitat moodle/site:accessallgroups (normalment estudiants o professorat no editor, etc.) només podran veure les respostes dels grups als que pertanyen.

Tots els participants inclou respostes de tothom que ha contestat l'enquesta, incloent les persones que no pertanyen a cap grup. Membres de: Tots els grups inclou respostes de la gent que pertany a qualque grup. I Persones sense cap grup s'explica tot sol.

Important.- Després d'haver utilitzat la llista desplegable per seleccionar un "conjunt de persones" de les quals es mostra les seves respostes, el "filtre" roman actiu per a les següents operacions: Descarregar en format text, Esborrar TOTES les respostes i Visualitzar per resposta. Per exemple, si heu seleccionat mostrar TOTES les respostes per al grup "Ocells" i després pitjau la pestanya Esborrar TOTES les respostes, un missatge de confirmació vos demanarà que confirmeu que voleu esborrar totes les respostes per als membres del grup "Ocells" i només les respostes d'aquests participants. Nota.- Si un participant pertany a més d'un grup, llavors si esborrau totes les respostes per a un dels grups als quals pertany, evidentment esborrareu les seves respostes de tots els altres grups als quals pertany.
