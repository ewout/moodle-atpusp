Ordonner les réponses

Sur l'écran qui affiche Toutes les réponses, vous pouvez ordonner les choix d'après le nombre de réponses (dans la colonne Moyenne ou Rang moyen) pour les 4 types de questions suivantes.

  • choix unique bouton radio ;
  • choix unique liste déroulante ;
  • réponses multiples (cases à cocher) ;
  • échelle de Likert.

Lorsque vous arrivez sur la page Toutes les réponses, par défaut toutes les réponses de ces 4 types de questions sont listées dans l'ordre où elles sont été entrées par le créateur des questions. Vous pouvez alors les ordonner en ordre ascendant ou descendant en cliquant sur les liens ad hoc.
