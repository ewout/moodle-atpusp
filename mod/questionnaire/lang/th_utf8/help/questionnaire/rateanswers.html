หมายเหตุ

คำถามประเภทนี้สามารถแสดงตัวเลือกได้ 2 แบบ ปกติจะแสดงแบบ Likert ระดับคะแนน (1...5) ถ้าคุณ ตั้งชื่อ แทนตัวเลข คุณจะต้องระบุชื่อในรายการ คำตอบ (ดู ตัวอย่างที่ 3 ด้านบน) แต่ละบรรทัดจะหมายถึง คะแนน "ตัวเลือก", ตามด้วย เครื่องหมายเท่ากับ = ตามด้วย ข้อความตัวเลือกที่ต้องการ

ตัวเลขหน้าชื่อตัวเลือกจะเป็นตัวบอกคะแนน ซึ่งระบบไม่ได้นำไปใช้งาน คุณอาจตั้ง 0 = ไม่เคย, 0 = บางครั้ง, 0 = บ่อย ก็ได้ แต่จะต้องเริ่มต้นด้วย เครื่องหมายเท่ากับ = .

ถ้า จำนวนตัวเลือก ในรายการคำตอบ ต่างจากตัวเลขที่ระบุ ในช่อง จำนวนคำตอบ ระบบจะปรับตัวเลขให้ใหม่ ถ้าต้องการใช้คะแนนแบบ Likert คุณต้องใส่ชื่อตัวเลือก แล้วตามด้วย ตัวเลือก ต่อไป เช่น "การเรียนเน้น...", "สิ่งที่เรียนสำคัญที่สุด", ตามตัวอย่างที่ 3 ด้านบน).

ตัวอย่าง 3b แสดงการ "ซ่อน" ตัวเลือก เมื่อทำการส่งออกแบบสอบถาม ข้อความจะสั้นลง ถ้าไม่เคยส่งออกข้อมูลก็ไม่ต้องสนใจเรื่องนี้ การซ่อนชื่อจะใช้ได้กับคำถามทุกประเภท ยกเว้น Osgood ซึ่งสั้นอยู่แล้ว
