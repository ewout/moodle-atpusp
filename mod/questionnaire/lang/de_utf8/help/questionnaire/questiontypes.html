Datum
Nutzen Sie das Tag/Monat/Jahr Format z.Bsp.:. für März 14er, 1945: 14/3/1945

Nutzen Sie diesen Fragetyp wenn Sie erwarten, dass die Antwort ein korrekt formatiertes Datum ist. Das Format hängt von der Sprache ab, die die befragte Person eingestellt hat. Zum Beispiel: 4/21/2007 (US); 21/4/2007 (UK); 21-4-2007 (France); usw. Ein Beispiel wird in der Befragung angezeigt. Wenn ein "unmögliches" oder falsch formatiertes Datum eingegeben wird, wird es entweder umgeschrieben oder (wenn möglich) umformatiert oder es wird eine Fehlermeldung angezeigt. Um ein Datum korrekt in ein Arbeitsblatt von z.Bsp.: Excel exportieren zu können, muß die befragte Person ein Datum zwischen 1902 und 2037 eingeben. Wenn ein Datum außerhalb dieser Zeitspanne erwartet wird, dann nutzen Sie bitte den Textbox-Fragentyp. Der Fragentyp Datum akzeptiert Eingaben von einem Monat und einem Jahr (z.Bsp.: 12/2008 für Dezember 2008) oder auch nur ein Jahr (z.Bsp.: 2008). Allerdings werden solche unvollständige Daten automatisch in vollständige (dd/mm/yy) umgewandelt, da Sie sonst nicht kompatibel für Arbeitsblätter sind: z.Bsp.: 12/2008 -> 01/12/2008 und 2008 -> 01/01/2008. Sie müssen diese typische Verhaltensweise Ihren Nutzer/innen eventuell vorher in der Befragung erklären.

Dropdown Box

Der einzige Grund warum eine Dropdown-Box in manchen Fällen eventuell Radiobuttons vorzuziehen wäre, ist die Länge der Liste der Optionen. Mit einer Dropdown-Box kann Platz auf dem Bildschirm gespart werden.

Label

Numerisch

Nutzen Sie diesen Fragetyp, wenn die Antwort eine korrekt formatierte Zahl ist. Indem Sie die Optionen Max. erlaubte Ziffern und Anzahl der Dezimalstellen nutzen, können Sie die Länge und Dezimalstellen der Zahlen definieren, die eingetragen werden sollen.
