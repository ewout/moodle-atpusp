Parameter der "Checkbox" Frage

Nutzen Sie diesen Parameter um der befragten Person das Anklicken von mindestens Min. Boxen und maximal Max. Checkboxen vorzugeben. Um eine exakte Anzahl von anzuklickenden Checkboxen vorzugeben, setzen Sie Min. und Max. auf den gleichen Wert. Wenn nur ein Mindest- oder Maximalwert gewünscht ist, dann lassen Sie einfach den anderen Wert auf der Voreinstellung 0. Wenn Sie die Min. oder Max. Werte auf etwas anderes als die Voreinstellung 0 setzen wird eine Warnung ausgegeben wenn die befragte Person die Vorgaben nicht befolgt. Sie sollten der befragten Person alle Vorgaben im einführenden Text der Befragung erklären oder im Text der jeweiligen Frage.
