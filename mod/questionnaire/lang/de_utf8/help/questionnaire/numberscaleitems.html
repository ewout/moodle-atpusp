Bewertungsskala - Frageparameter

  • Die Anzahl der Skaleneinträge ist die Anzahl der Elemente, die in Ihrer Bewertungsskala genutzt werden. Sie werden üblicherweise Werte von 3 bis 5 benutzen. Voreinstellung: 5.
  • Typen von Bewertungsskalen
    • Normal (Voreinstellung)
    • k.A. Zeile : Wählen Sie dies, wenn Sie wünschen, dass eine k.A.-Zeile auf der rechten Seite der Zeile mit den Einträgen in Ihrer Bewertungsskala hinzugefügt werden soll.
    • Keine doppelte Wahlmöglichkeit: Wählen Sie dies wenn Sie doppelte Wahlmöglichkeiten in jeder Bewertungszeile verhindern wollen.
      • Dies ist nützlich wenn Sie möchten, dass die befragte Person eine Anzahl von Elementen auf einer Skala von 1 bis n bewertet und jede Bewertung eindeutig sein soll. Beispiel: Elemente A, B und C nach Präferenz bewerten. Akzeptiert wird: A1, B3 und C2 oder A3, B2 und C1, aber es wird für die befragte Person nicht möglich sein einzutragen: A1, B1, C2 oder A1, B2, C2, usw.
      • Diese Option wird genutzt in Verbindung mit der Option der benannten Werte. Keine doppelte Wahlmöglichkeit kann ebenfalls nützlich sein wenn Sie möchten, dass die befragte Person Einträge mit benannten Werten verbindet, z.Bsp.: die Farben Rot, Blau, Gelb mit physiologischen Antworten: Aufregung,Ruhe, Konzentration wo eine Farbe nur einer physiologischen Antwort zugewiesen werden kann.
    • Osgood: Wählen Sie dies, um eine Frage zu erstellen nach dem Typ Osggod's semantisches Differenziall Typ. Dieser Parameter muss in Verbindung mit der Option der benannten Werte genutzt werden.
