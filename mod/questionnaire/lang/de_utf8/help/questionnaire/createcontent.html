Neue Befragung - Inhalt - Optionen

Zur Erstellung des Inhalts einer neuen Befragung wählen sie eine der Radio-Button-Optionen. "Neu erstellen" ist die Voreinstellung.

    Neue Befragungen können auf drei verschiedenen Wegen erstellt werden:
  1. Neu erstellen:
    Mit dieser Auswahl erstellen Sie eine neue Befragung von Anfang an.
  2. Bestehende kopieren:
    Bei dieser Auswahl wird der Inhalt einer bestehenden Befragung in eine neue Befragung kopiert. Sie haben die Auswahl zwischen bereits erstellten Vorlagen (Templates) oder im Kurs vorhandenen Befragungen. Mit einem Klick auf den Namen erhalten Sie eine Vorschau.
  3. Öffentliche nutzen:
    Wenn eine Befragung bereits existiert (in einem anderen Kurs auf der gleichen Moodle-Installation) und die Einstellungen auf "öffentlich" gesetzt wurden, können Sie diese "öffentliche" Befragungen in Ihrem eigenen Kurs nutzen. Die Anzahl an Optionen ist allerdings eingeschränkt und Sie können solche Befragungen nicht editieren und die Antworten nicht ansehen.
    Beispiel Wenn eine öffentliche Befragung in Kurs A erstellt wurde, kann sie in den Kursen B, C, ... "genutzt" werden. Alle Antworten der Kurse A, B, C, ... werden in der öffentlichen Befragung in Kurs A (dem Orginalkurs) gesammelt und können dort von der Person die sie erstellt hat (admin oder Lehrer/in) angesehen werden.
