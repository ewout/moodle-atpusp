Appearance

Notes: A web page displayed under the video window. Supports rich text, links, images, plugins, Flash, etc.

Width: 0 - 9999 (% values are also possible)

Height: 0 - 9999

Skin: change the design of the video player interface

Poster Image: JPG, GIF, PNG or SWF to display before video playback starts (Behaviour > Auto Start must be set to "false")

Show Icons: show pause/play/streaming icons in centre of video window

Control Bar:

  • none = no playback controls
  • bottom = playback controls under the video window
  • over = playback controls on top of the video window

Control Bar Colors:

  • Back Color - control bar background color
  • Light Color - control bar icons and text color
  • Front Color - control bar video position scrubber and volume control color
  • Screen Color - video window background color

Smoothing: smooths out pixels in video image (leave as "true" unless users have very slow computers)

Quality: overall display quality of Flash Player plugin (leave as "best" unless users have very slow computers)

Colors are hexidecimal values - e.g. black = 000000, white = FFFFFF, red = FF0000, green = 00FF00 and blue = 0000FF
