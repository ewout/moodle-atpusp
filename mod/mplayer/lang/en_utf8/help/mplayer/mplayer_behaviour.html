Behaviour

Auto Start: start playing video as soon as page loads

Full Screen: allow users to switch video to take up the whole screen

Stretching:

  • none = no stretching
  • uniform = largest possible while maintaining aspect ratio, no cropping
  • exactfit = ignore aspect ratio
  • fill = fill the player window, some cropping may occur

Volume: audio volume (users adjust it on control bar)

Mute: mute audio (users can turn it on on control bar)

Start: start position of video in seconds (with progressive download, video must download to the start position before playback can start)

Buffer Length: number of seconds of video to pre-download before playing (set this to higher values for users with slow/unreliable/intermittent internet connections)

Plugins: add additional functions to the video player. See: http://www.longtailvideo.com/addons/plugins
