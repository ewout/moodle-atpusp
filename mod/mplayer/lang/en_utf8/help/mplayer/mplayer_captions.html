Captions

Transparent Background: If set to true, the plugin will render a semi-transparent background instead of a black outline.

Captions File: supports SMIL TimedText.xml and SubRip.srt*

Font Size: size of captions text

Show Captions: show captions by default (users can switch them on or off)

* Subtitles displayed using the captions plugin are much clearer and easier to read than embedded captions and you have more control over their appearance. Here's a useful tool for extracting embedded subtitles from video files: http://zuggy.wz.cz/

More details about this plugin: http://developer.longtailvideo.com/trac/wiki/PluginsCaptions
