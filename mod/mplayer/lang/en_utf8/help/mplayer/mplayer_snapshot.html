Snapshot

WARNING! Experimental plugin which is not secure. This should not be deployed on a publicly accessible server.

Bitmap: either sends bitmap image to server (implemented) or sends position of video frame to server for server-side snapshot (not implemented)

Script: script to receive and write snapshots on server

For more information see: http://developer.longtailvideo.com/trac/wiki/PluginsSnapshot
