Search Bar

Color: color of search button (hexidecimal number)

Label: text on search button

Position:

  • none = no search bar
  • top = above video window
  • bottom = under video window

Script: default is YouTube.com API. Custom search scripts can also be used. Contact your IT support for more details.

NB: Requires playlist to display results. If no playlist parameters are set, it creates a default playlist.

For more information see: http://developer.longtailvideo.com/trac/wiki/PluginsSearchbar
