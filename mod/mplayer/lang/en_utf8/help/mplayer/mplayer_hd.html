HD Video

Bitrate: This only needs to be set for RTMP streaming to enable automated bandwidth switching

HD Video File: alternative HD video file

Switch to HD on Full Screen: when user clicks on full screen button, stream switches to HD version

Default HD: play HD video by default

With progressive download, video will restart when switching between HD and normal streams.

For more information see: http://developer.longtailvideo.com/trac/wiki/PluginsHd
