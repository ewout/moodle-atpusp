What can be Monitored?

At this stage, a Progress Bar block can be used to monitor an individual student's progress in the following set activities and resources.

  • Assignment submissions
  • Forum postings
  • Quiz attempts
  • Choice answers
  • Feedback survey responses
  • Resources, specifically...
    • Text and HTML pages viewed, and
    • Links to files or webpages clicked

Monitored items must be visible to be shown in the progress bar. Hiding an activity or resource will hide the event in the progress bar.
