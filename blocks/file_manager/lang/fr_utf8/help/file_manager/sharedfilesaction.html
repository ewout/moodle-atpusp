Gestionnaire de fichiers

Actions sur les partages

Vous pouvez supprimer les partages qui vous ont été attribués. Pour cela, sélectionnez le ou les partages que vous voulez supprimer puis activez l'option "Supprimer" dans la liste déroulante.
Cette action n'efface pas le fichier en tant que tel, mais vous supprime l'accès seulement. Si vous désirez pouvoir y accèder à nouveau, il faut que le propriétaire de la ressource vous la partage à nouveau.
