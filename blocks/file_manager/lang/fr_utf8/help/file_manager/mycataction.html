Actions sur les catégories

Partager avec d'autres utilisateurs
Vous pouvez partager une ou plusieurs catégories en les cochant puis en sélectionnant l'action dans la liste déroulante "Partager avec d'autres utilisateurs".

Supprimer
Cochez la ou les catégories que vous souhaiter supprimer puis sélectionnez l'action dans la liste déroulante "Supprimer".
