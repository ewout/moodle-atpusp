Montrer les partages du cours uniquement

Par défaut, le gestionnaire de fichier vous présente les partages qui vous ont été attribués dans le cours depuis lequel vous avez appelé cette fonction.

Montrer tous les partages du site

Vous pouvez par ailleurs visualiser tous les partages qui vous été attribués sur la totalité du site, peu importe dans quel cours vous vous trouvez présentement.
