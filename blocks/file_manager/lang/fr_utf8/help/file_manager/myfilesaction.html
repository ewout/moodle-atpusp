Gestionnaire de fichiers

Actions sur les ressources

Déplacer vers un autre répertoire
Sélectionnez la ou les ressources que vous souhaitez déplacer, puis sélectionnez dans la liste déroulante l'action "Déplacer vers un autre répertoire". Ensuite déplacez-vous dans l'arborescence jusqu'à destination et cliquez sur le bouton "Déplacer les fichiers ici".

Associer à une catégorie
Sélectionnez la ou les ressources au(x)quelle(s) vous souhaitez associer une catégorie, puis sélectionnez dans la liste déroulante l'action "Associer à une catégorie". Ensuite sélectionnez la nouvelle catégorie puis cliquez sur le bouton "Associer".

Créer une archive ZIP
Sélectionnez la ou les ressources à partir de(s) la(les)quelle(s) vous souhaitez créer une archive ZIP, puis sélectionnez dans la liste déroulante l'action "Créer une archive ZIP". Ensuite sélectionnez le format souhaité : ZIP Standard ou ZIP Moodle.

Supprimer
Cochez la ou les ressources que vous souhaiter supprimer puis sélectionnez l'action dans la liste déroulante "Supprimer".
