File Manager

Actions on links

Move to another folder
Select links or files that you want to move, then select the action "Move to another folder" in the list. Literaly move to the final destination and click on the button "Move files here".

Assign a category
Select links or files that you want to assign a category, then select the action "Assign a category" in the list and select the category name before clicking on the button "Assign".

Create ZIP archive
Select links or files that you want to add to the a new archive, then select the action "Create ZIP archive" and finaly select the Zip style : Standard or Moodle Zip.

Delete
Select files or links that you want to delete and select the action "Delete" in the list.
