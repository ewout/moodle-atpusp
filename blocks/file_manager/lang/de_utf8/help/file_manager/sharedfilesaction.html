Dateimanager

Aktionen für bereitgestellte Dateien/Links

Sie können die Verweise zu den Ihnen zur Verfügung gestellte Dateien/Links löschen. Wählen Sie dafür die Datei(en) bzw. den Link oder die Links, die Sie löschen wollen, aus und klicken Sie "Löschen" in der Liste.
Diese Aktion löscht nicht die Dateien oder Links aus dem Dateimanager. Sie löschen nur den Zugang zu den Ihnen zur Verfügung gestellten Dateien oder Links. Wenn Sie erneut Zugang wünschen, muss der Eigentümer der Dateien bzw. Links Ihnen diese erneut zur Verfügung stellen.
