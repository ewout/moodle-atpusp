Dateimanager

Aktionen für Links

Verschieben in anderen Ordner
Wählen Sie die Links aus, die Sie verschieben wollen. Wählen Sie dann die Aktion "In anderen Ordner verschieben" in der Liste. Gehen Sie dann zum Ziel und klicken Sie "Hierher verschieben".

Kategorie zuweisen
Wählen Sie die Links aus, denen Sie eine Kategorie zuweisen wollen. Wählen Sie dann die Aktion "Kateogrie zuweisen" in der Liste und wählen Sie die Kategorie aus, bevor Sie "Zuweisen" klicken.

ZIP erstellen
Wählen Sie die Links oder Dateien, die Sie komprimieren wollen. Wählen Sie dann die Aktion "ZIP-Datei erstellen" und schliesslich das ZIP-Format: Moodle-ZIP oder Standard-ZIP.

Löschen
Wählen Sie die Links, die Sie löschen wollen und wählen sie schliesslich "Löschen" in der Liste.
