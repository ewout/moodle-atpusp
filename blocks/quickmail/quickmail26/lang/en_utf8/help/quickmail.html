Quickmail v2.5

Quickmail v2.5 allows you to send a message to one or more recipients in the Potential Recipient(s) list.
You can add recipients one at a time by selecting a name from the Potential Recipient(s) list and clicking the Add button.
You can add multiple recipients at the same time using any of the following methods:

  • click and drag the cursor to select multiple recipients in sequential order
  • hold the shift key and click to select multiple recipients in sequential order
  • hold the control key (Windows) or command key (Macintosh) and click to select multiple recipients in non-sequential order

This message will be sent immediately after you click the Send Email button, even to those Moodle users who have chosen to receive discussion forum messages in digest format (i.e., once per day).

Some of the new features in Quickmail v2.5 include:

  • You can now limit your choices based on class role. For instance: If you only want ot send to students, click students in the drop down list above your sections and groups and only students will be selected
  • Chose email recipients based on group and section membership
  • You no longer MUST enter anything in the message area
  • Adding multiple attachments has been added to the system
  • You can now chose to receive sent mail receipts or not (The default is no)

We have more planned for Quickmail v3, so please send feedback to elearning@lsu.edu for future enhancement requests.
